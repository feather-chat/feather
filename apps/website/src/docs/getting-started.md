# Getting Started

Enzyme is a self-hosted team chat application. This guide walks you through creating your account, joining a workspace, and sending your first message.

## Creating an Account

To use Enzyme, you need an account on your team's Enzyme instance. Navigate to the URL your administrator has shared and click **Sign up**. You'll need:

- **Email address** — used for login and email notifications
- **Password** — minimum 8 characters
- **Display name** — how you'll appear to other users

After registering, you'll receive a verification email. Click the link to verify your address. You can start using Enzyme immediately, but some features (like email notifications) require a verified email.

## Joining a Workspace

Workspaces are separate team spaces within Enzyme. You can join a workspace in two ways:

1. **Invite link** — An admin shares an invite URL with you. Click it to join with the role specified in the invite.
2. **Create your own** — After logging in, you can create a new workspace and invite others.

When you join a workspace via invite:

- You're automatically added to the **#general** channel.
- Direct message conversations are created with a few existing members so you can start chatting right away.

## Navigating the Interface

The Enzyme interface has three main areas:

- **Workspace switcher** (far left) — Switch between workspaces you belong to.
- **Sidebar** (left) — Browse channels, direct messages, and starred items. Access your threads and unreads.
- **Main area** (center) — Read and write messages in the selected channel or conversation.

### Command Palette

Press **Cmd+K** (Mac) or **Ctrl+K** (Windows/Linux) to open the command palette. From here you can:

- Jump to any channel or direct message
- Find people in your workspace
- Search messages
- Create channels, start DMs, and more

### Quick Compose

Start typing anywhere (when no input is focused) and the message composer will automatically focus and capture your keystrokes.

## Sending Your First Message

1. Select a channel from the sidebar (start with **#general**).
2. Click the message input at the bottom of the screen, or just start typing.
3. Type your message and press **Enter** to send.

See [Message Formatting](/docs/messages/) for details on bold, italic, code blocks, mentions, and more.
