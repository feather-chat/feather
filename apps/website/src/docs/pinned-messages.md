# Pinned Messages

Pin important messages to make them easy to find later. Pinned messages are visible to all channel members.

## Pinning a Message

Hover over a message and select **Pin** from the message actions menu. A system message is posted in the channel to let others know the message was pinned.

Members, admins, and owners can pin messages. Guests cannot. See [Permissions & Roles](/docs/permissions/) for details.

## Viewing Pinned Messages

Click the pin icon in the channel header to view all pinned messages for that channel. Pins are displayed in the order they were pinned.

## Unpinning

Select **Unpin** from the message actions menu on a pinned message. The same permissions apply as for pinning.

## Limits

Each channel can have a maximum of **50 pinned messages**. If the limit is reached, unpin an older message before pinning a new one.
