# Keyboard Shortcuts

Enzyme supports keyboard shortcuts for common actions. On Mac, use **Cmd**; on Windows and Linux, use **Ctrl**.

## Global Shortcuts

| Shortcut                           | Action               |
| ---------------------------------- | -------------------- |
| **Cmd+K** / **Ctrl+K**             | Open command palette |
| **Cmd+Shift+F** / **Ctrl+Shift+F** | Search messages      |

## Command Palette

The command palette (**Cmd+K**) is the fastest way to navigate. Start typing to:

- **Jump to a channel** — type a channel name
- **Open a DM** — type a person's name
- **Search messages** — type a search query and select "Search messages for..."
- **Run actions** — type "create channel", "invite", "threads", "scheduled", etc.

Use **arrow keys** to move through results and **Enter** to select.

## Composing Messages

| Shortcut        | Action                                             |
| --------------- | -------------------------------------------------- |
| **Enter**       | Send message                                       |
| **Shift+Enter** | New line                                           |
| Any letter      | Auto-focus the composer (when no input is focused) |

## Autocomplete in Composer

While typing a message, the following triggers show autocomplete suggestions:

| Trigger | Autocomplete      |
| ------- | ----------------- |
| `@`     | Mention a user    |
| `#`     | Link to a channel |
| `:`     | Insert an emoji   |

Use **arrow keys** and **Enter** or **Tab** to select from suggestions.
