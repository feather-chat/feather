# Direct Messages

Direct messages (DMs) are private conversations that live outside of channels.

## One-on-One DMs

To start a DM with someone:

1. Open the command palette (**Cmd+K**) and type their name, or
2. Click the **+** next to "Direct Messages" in the sidebar.

DMs between the same two people always use the same conversation — starting a new DM with someone you've already messaged opens the existing thread.

## Group DMs

Group DMs support 3 to 8 participants. To create one:

1. Click the **+** next to "Direct Messages" in the sidebar.
2. Select multiple people.

Group DMs display the names of all participants in the sidebar.

## DMs vs Channels

| Feature       | DMs                         | Channels                     |
| ------------- | --------------------------- | ---------------------------- |
| Participants  | Fixed when created          | Members can join/leave       |
| Visibility    | Only participants can see   | Public or private            |
| Notifications | Default to **all** messages | Default to **mentions** only |
| Archiving     | Cannot be archived          | Can be archived              |

## Converting to a Channel

Group DMs can be converted into a full channel. This gives the conversation a name, makes it appear in the channel list, and allows new members to join. Any member of the group DM can perform the conversion (except guests).

After conversion, the full message history is preserved in the new channel.
